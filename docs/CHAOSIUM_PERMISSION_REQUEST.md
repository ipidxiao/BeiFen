# Chaosium Permission Request Template

Use this template to request written permission from Chaosium Inc. to redistribute JSON scenario adaptations of their official adventures within the CoC Engine. **Only the rights holder can grant this permission** — the engine maintainers cannot obtain it programmatically.

---

## Before You Send

1. Read the current [Fan Material Policy](https://www.chaosium.com/fan-material-policy/) (revised October 13, 2023).
2. Understand that the Fan Policy **explicitly excludes** apps, downloadable software, and VTTs — a separate **Commercial License** may be required.
3. Prepare a brief description of your project, audience, and whether distribution would be free or paid.
4. Do **not** redistribute Chaosium proprietary scenario text or JSON until you receive written approval.

## After You Receive a Response

- **If approved:** Keep a copy of the written permission. Update `remote_catalog.mjs` only after legal review; do not commit proprietary JSON without explicit approval documented in your project.
- **If denied or no response:** Continue using link-only catalog entries and user self-import from officially downloaded PDFs.
- **If referred to Commercial Licensing:** Follow [Commercial Licensing](https://www.chaosium.com/commercial-licensing/) requirements (formal proposal + business plan).
